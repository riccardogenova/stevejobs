COSA è HTML

HTML, acronimo di HyperText Markup Language (Linguaggio di Markup per IperTesto),
è il linguaggio standard utilizzato per creare e strutturare pagine web.
È uno dei fondamenti del World Wide Web e consente agli sviluppatori
di definire la struttura e il contenuto delle pagine web,
inclusi testi, immagini, collegamenti ipertestuali e altri elementi multimediali.

FUNZIONI PRINCIPALI DI HTML

**Linguaggio di Markup**: HTML è un linguaggio di markup, il che significa che utilizza una serie di tag
per definire la struttura e il significato del contenuto all'interno di una pagina web.
Ogni tag HTML è racchiuso tra parentesi angolari `< >` e fornisce istruzioni
al browser su come visualizzare il contenuto.

**Struttura della Pagina**: HTML definisce la struttura di base di una pagina web.
Ogni documento HTML inizia con un elemento html che contiene due sezioni principali: head e body.
La sezione head contiene informazioni di intestazione come il titolo della pagina,
i metadati e i collegamenti a fogli di stile esterni o script.
La sezione body contiene il contenuto visibile della pagina,
come testo, immagini, tabelle e formulari.

**Elementi e Tag**: Gli elementi HTML sono i mattoni di base con cui viene costruita una pagina web.
Ogni elemento è definito da un tag HTML, che specifica il tipo di elemento e
può avere attributi che forniscono ulteriori informazioni sull'elemento.
Ad esempio, il tag p viene utilizzato per definire un paragrafo di testo,
mentre il tag img viene utilizzato per inserire un'immagine nella pagina.

**Struttura Gerarchica**: Gli elementi HTML sono solitamente organizzati in una struttura gerarchica,
con alcuni elementi contenuti all'interno di altri. Ad esempio, un paragrafo p può essere contenuto
all'interno del corpo body della pagina,
mentre il corpo è contenuto all'interno dell'elemento radice html.

**Semantica**: HTML fornisce una serie di tag semantici che descrivono
il significato del contenuto piuttosto che solo l'aspetto.
Questi tag includono header, footer, nav, article, section e altri,
che forniscono informazioni sulla struttura e il significato del contenuto,
migliorando l'accessibilità e l'indicizzazione da parte dei motori di ricerca.

**Compatibilità e Standardizzazione**: HTML è un linguaggio standardizzato e supportato da tutti i principali browser web.
Le specifiche e gli standard per HTML sono sviluppati e mantenuti
dal World Wide Web Consortium (W3C),
un'organizzazione internazionale che stabilisce le linee guida per lo sviluppo del web.

HTML_SHEET

**`!DOCTYPE`**: Definisce il tipo di documento HTML.

```html

!DOCTYPE html

```
**`html`**: Definisce l'inizio e la fine di un documento HTML.

```html

html

!-- Contenuto della pagina --

/html

```
**`head`**: Contiene informazioni di intestazione come il titolo della pagina,
i metadati e i collegamenti a fogli di stile esterni o script.
```html

head

title Titolo della Pagina /title

meta charset="UTF-8"

meta name="description" content="Descrizione della pagina"

link rel="stylesheet" href="styles.css"

/head

```
**`title`**: Definisce il titolo della pagina visualizzato nella barra del browser.

```html

title Il mio sito web /title

```
**`body`**: Contiene il contenuto visibile della pagina web.

```html

body

h1 Benvenuto! /h1

p Questo è il mio primo sito web. /p

/body
```
**`h1`, `h2`, `h3`, ..., `h6`**: Definisce i livelli di intestazione (titoli) con diversi livelli di importanza.

```html

h1 Titolo Principale /h1

h2 Titolo Secondario /h2

h3 Titolo Tersiario /h3

```
**`p`**: Definisce un paragrafo di testo.

```html

p Questo è un paragrafo di testo. /p

```
**`a`**: Definisce un collegamento ipertestuale (link)
a un'altra pagina web o a un'altra sezione della stessa pagina.

```html

a href="https://www.example.com">Visita il sito web /a

```
**`img`**: Inserisce un'immagine nella pagina web.

```html

img src="immagine.jpg" alt="Descrizione dell'immagine"

```
**`ul`**: Definisce una lista non ordinata.

```html

ul

li Elemento 1 /li

li Elemento 2 /li

li Elemento 3 /li

/ul

```
**`ol`**: Definisce una lista ordinata.

```html

ol

li Primo elemento /li

li Secondo elemento /li

li Terzo elemento /li

/ol

```
**`table`**: Definisce una tabella.

```html

table

tr

td Cella 1 /td

td Cella 2 /td

/tr

tr

td Cella 3 /td

td Cella 4 /td

/tr

/table

```
**`form`**: Definisce un modulo per la raccolta di input utente.

```html

form action="/submit" method="post"

label for="username">Nome utente: /label

input type="text" id="username" name="username"

input type="submit" value="Invia"

/form

```
**'meta'**: Fornisce metadati sul documento HTML come il set di caratteri,
la descrizione e le parole chiave per i motori di ricerca. **form**: Definisce un modulo per la raccolta di input utente.
**input**: Definisce un campo di input all'interno di un modulo.
**textarea**: Definisce un campo di testo multi-riga all'interno di un modulo.
**button**: Definisce un pulsante cliccabile.
**label**: Definisce l'etichetta per un elemento di input.
**select**: Definisce un menu a discesa (dropdown).
**option**: Definisce una voce di menu a discesa all'interno di un select.
**iframe**: Inserisce una finestra di visualizzazione di un'altra pagina web all'interno della pagina corrente.
**script** : Definisce uno script client-side, come JavaScript.
**style**: Definisce regole di stile CSS all'interno del documento HTML.
**audio**: Inserisce un file audio nella pagina web.
**video**: Inserisce un file video nella pagina web.
**div**: Definisce una divisione o una sezione generica all'interno di un documento.
**span**: Definisce una porzione di testo in linea o una sezione generica all'interno di un documento.
**article**: Definisce un articolo indipendente.
**aside**: Definisce un contenuto a parte, come una barra laterale.
**figure**: Definisce un contenuto multimediale con una didascalia.
**figcaption**: Definisce la didascalia per un contenuto multimediale.

CONSIDERAZIONI FINALI

In sintesi, HTML è fondamentale per la creazione di pagine web,
poiché fornisce una struttura organizzata e semantica per il contenuto della pagina,
consentendo ai browser web di interpretare e visualizzare il contenuto in modo coerente per gli utenti.
