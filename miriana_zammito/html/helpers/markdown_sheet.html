FORMATTAZIONE DEL TESTO

  • Grassetto: ** testo **, __ testo __
  • Corsivo: * testo *, _ testo _
  • Grassetto e corsivo: *** testo ***
  • Barrato: ~~ testo ~~
  • Evidenziato: `testo`

    • LISTE

    Elenco puntato:

  • * Elemento 1
  • - Elemento 2
  • + Elemento 3

    • Elenco numerato:

  • 1. Elemento A
  • 2. Elemento B

    • Liste di controllo

  • [x] Elemento completato
  • [ ] Elemento da fare

    • LINK ED IMMAGINI

  • Link: [nome link](URL)
  • Immagine: ![nome immagine](URL immagine)

  • citazione: >testo
  • blocco di codice/codice: ```
