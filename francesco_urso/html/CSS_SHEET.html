Selettori CSS

Selettori divisi per tipologia:

  • Selettori di tipo
  • Selettori di classe
  • Selettori di ID
  • Selettori di figlio
  • Selettori discendenti
  • Selettori di figlio diretto
  • Selettori di attributo
  • Selettori di attributo con valore specifico
  • Selettori di pseudo-classe

Selettori di Tipo

I selettori di tipo selezionano tutti gli elementi HTML di un certo tipo. Ad esempio, il selettore p seleziona tutti gli elementi <p> presenti nel documento HTML.
p {
/* Regole CSS *
}

Selettori di Classe

I selettori di classe selezionano gli elementi HTML che hanno un determinato valore nell'attributoclass. In CSS, i selettori di classe iniziano con il carattere punto . seguito dal nome della classe.
.nome-classe {
/* Regole CSS */
}

Selettori di ID

I selettori di ID selezionano un elemento HTML con un ID specifico. In CSS, i selettori di ID iniziano con il carattere cancelletto # seguito dal nome dell'ID.
#nome-id {
/* Regole CSS */
}

Selettori di figlio

I selettori di figlio selezionano gli elementi figlio di un elemento specifico. Si utilizza lo spazio per separare il selettore dell'elemento genitore dal selettore dell'elemento figlio. Ad esempio, il selettore ul li selezionerà tutti gli elementi <li> che sono figli di un elemento <ul>.
ul li {
/* Regole CSS */
}

Selettori discendenti

I selettori discendenti selezionano gli elementi che sono discendenti diretti o indiretti di un altro elemento, indipendentemente dal livello di annidamento. Si utilizza uno spazio per separare il selettore dell'elemento genitore dal selettore dell'elemento discendente. Ad esempio, il selettore div p selezionerà tutti gli elementi <p> che sono discendenti di un elemento <div>.
div p {
/* Regole CSS */
}

Selettori di figlio diretto

I selettori di figlio diretto selezionano solo gli elementi che sono figli diretti di un elemento genitore specifico, senza considerare gli elementi discendenti ulteriori. Si utilizza il carattere > per indicare un figlio diretto. Ad esempio, il selettore div > p selezionerà solo gli elementi <p> che sono figli diretti di un elemento <div>.
div > p {
/* Regole CSS */
}

Selettori di attributo

I selettori di attributo selezionano gli elementi che hanno un attributo specifico o che soddisfano determinate condizioni sugli attributi. Ad esempio, il selettore [type] seleziona tutti gli elementi che hanno un attributo type, indipendentemente dal suo valore.
[attributo] {
/* Regole CSS */
}

Selettori di attributo con valore specifico

I selettori di attributo con valore specifico selezionano gli elementi che hanno un attributo specifico con un valore corrispondente. Ad esempio, il selettore [type="text"] seleziona tutti gli elementi che hanno un attributo type con valore "text".
[attributo="valore"] {
/* Regole CSS */
}

Selettori di pseudo-classe

Le pseudo-classi selezionano elementi in uno stato specifico o in relazione ad altri elementi, come ad esempio un link non visitato, un elemento al passaggio del mouse, il primo elemento di un tipo e così via. Le pseudo-classi iniziano con due punti : seguiti dal nome della classe pseudo.
selettore:pseudo-classe {
/* Regole CSS */
}

Proprietà CSS

Proprietà di Testo

  • color: Specifica il colore del testo.
  • font-family: Specifica il tipo di carattere.
  • font-size: Specifica la dimensione del testo.
  • font-weight: Specifica il peso del carattere (normale, grassetto, ecc.).

Esempio:
p {
color: #333; /* Colore del testo (espresso in codice esadecimale) */
font-family: Arial, sans-serif; /* Tipo di carattere */
font-size: 16px; /* Dimensione del testo */
font-weight: normal; /* Peso del carattere (normale) */
}

Proprietà di Background

  • background-color: Specifica il colore dello sfondo.
  • background-image: Specifica un'immagine di sfondo.
  • background-position: Specifica la posizione dell'immagine di sfondo.

Esempio:
div {
background-color: #f0f0f0; /* Colore dello sfondo (grigio chiaro) */
background-image: url('path/to/image.jpg'); /* Immagine di sfondo */
background-position: center center; /* Posizione dell'immagine di sfondo (centro) */
}

Proprietà di Margine e Padding

  • margin: Specifica il margine esterno di un elemento.
  • padding: Specifica lo spazio interno di un elemento.

Esempio:
div {
width: 200px; /* larghezza dell'elemento */
height: 100px; /* altezza dell'elemento */
background-color: #f0f0f0; /* colore di sfondo */
margin: 20px; /* margine esterno di 20 pixel su tutti i lati */
padding: 10px; /* spazio interno di 10 pixel su tutti i lati */
}

Proprietà di Bordo

  • border: Specifica lo stile, il colore e lo spessore del bordo.
  • border-radius: Specifica il raggio degli angoli del bordo.

Esempio:
div {
width: 200px; /* larghezza dell'elemento */
height: 100px; /* altezza dell'elemento */
background-color: #f0f0f0; /* colore di sfondo */
border: 2px solid #333; /* bordo di 2 pixel di spessore, di colore nero */
border-radius: 10px; /* angoli arrotondati di 10 pixel */
}

Unità di Misura

Unità di Lunghezza

  • px: Pixel.
  • em: Relativo alla dimensione del carattere.
  • %: Percentuale rispetto al contenitore genitore.

Esempio:
.container {
width: 500px; /* larghezza del contenitore in pixel */
font-size: 16px; /* dimensione del carattere di base in pixel */
}

.box {
width: 200px; /* larghezza dell'elemento in pixel */
height: 100px; /* altezza dell'elemento in pixel */
font-size: 1.2em; /* dimensione del carattere dell'elemento relativa al font-size del contenitore */
margin: 20px; /* margine esterno dell'elemento in pixel */
padding: 10%; /* padding interno dell'elemento in percentuale rispetto alla larghezza */
}

Unità di Colore

  • Hexadecimal: #RRGGBB.
  • RGB: rgb(r, g, b).
  • RGBA: rgba(r, g, b, a) (con trasparenza).

Esempio:
.hex-color {
color: #ff0000; /* colore rosso in formato esadecimale */
background-color: #00ff00; /* colore verde in formato esadecimale */
}

.rgb-color {
color: rgb(0, 0, 255); /* colore blu in formato RGB */
background-color: rgb(255, 255, 0); /* colore giallo in formato RGB */
}

.rgba-color {
color: rgba(255, 0, 255, 0.5); /* colore magenta con 50% di trasparenza in formato RGBA */
background-color: rgba(0, 255, 255, 0.8); /* colore ciano con 80% di trasparenza in formato RGBA */
}

Eredità e Sovrascrittura

Inheritance Alcune proprietà vengono ereditate dagli elementi figlio.
Sovrascrittura Le regole CSS con maggiore specificità sovrascrivono quelle meno specifiche. Le dichiarazioni più recenti sovrascrivono quelle precedenti.

Guida utilizzo tramite Chrome

  1. Apri Google Chrome sul tuo computer.
  2. Nella barra degli strumenti del browser, fai clic su "Altro" (i tre punti verticali) nella parte superiore destra.
  3. Seleziona "Altri strumenti" e poi "Strumenti sviluppatore".
  4. Vai alla scheda "Console".

Dopo aver aperto la console, possiamo iniziare con alcuni esempi:

  • Per selezionare un elemento HTML, possiamo utilizzare document.querySelector().Ad esempio, se vogliamo selezionare il primo paragrafo <p> nella pagina, possiamo digitare:
    const primoParagrafo = document.querySelector('p');
    console.log(primoParagrafo);
  • Per modifica del testo di un elemento HTML, dopo aver selezionato un elemento, possiamo modificare il suo testo utilizzando innerText o innerHTML. Ad esempio, se vogliamo cambiare il testo del primo paragrafo in "Questo è il nuovo testo":
    primoParagrafo.innerText = "Questo è il nuovo testo";
  • Per la creazione di un nuovo elemento HTML, o anche creare nuovi elementi HTML, si utilizza document.createElement(). Ad esempio, se vogliamo creare un nuovo elemento <div> nella pagina, possiamo digitare:
    const nuovoDiv = document.createElement('div');
    console.log(nuovoDiv);
  • Per l'aggiunta di un nuovo elemento HTML alla pagina, possiamo aggiungerlo con appendChild() o insertBefore(). Ad esempio, se vogliamo aggiungere un nuovo elemento <div> alla fine del body:
    document.body.appendChild(nuovaDiv);
